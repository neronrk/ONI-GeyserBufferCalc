# 💧 Geyser Buffer Calc (RUS)

Мод для **Oxygen Not Included**, который добавляет расчётные данные в панель информации о гейзере. Полностью на русском языке.

## ✨ Что добавляет мод

При выборе гейзера в правой нижней панели (блок **Информация о потоке**) появляются новые строки:

| Параметр | Описание |
|----------|----------|
| **Средний поток при активности** | Сколько ресурса гейзер выдаёт в среднем, пока активен |
| **Буфер извержения** | Сколько нужно запасти, чтобы поддерживать поток во время пауз внутри активного цикла |
| **Общий средний поток** | Средний расход с учётом периодов покоя (за весь срок жизни) |
| **Буфер покоя** | Запас для поддержания базы во время долгой спячки гейзера |
| **Всего за жизнь** 🔥 | **Общий объём ресурса**, который гейзер выдаст от появления до исчезновения |

Также мод показывает тепловую мощность и потенциал охлаждения (если температура гейзера отличается от нормы).

## 📥 Установка

### Способ 1: Вручную
1. Скачай файлы из репозитория (кнопка **Code → Download ZIP**).
2. Распакуй содержимое в папку:

%USERPROFILE%\Documents\Klei\OxygenNotIncluded\mods\Local\GeyserBufferCalc\

3. В главном меню игры зайди в **Mods** → включи **Geyser Buffer Calc**.
4. Загружай сохранение — расчёты появятся автоматически.

### Способ 2: Через мод-менеджер
Если используешь **ONIModManager** или **Steam Workshop Downloader**, просто укажи путь к папке с модом.

## 🛠️ Сборка из исходников

Если хочешь собрать мод сам (например, чтобы изменить текст):

1. Убедись, что установлен [.NET Framework 4.x](https://dotnet.microsoft.com/download/dotnet-framework).
2. Открой папку с исходниками.
3. Запусти `build.bat`.
4. Готовый файл `AutomaticGeyserCalculation.dll` появится в папке.

> ⚠️ **Важно:** Не меняй `staticID` в `mod.yaml`, иначе игра не увидит мод.

## ⚙️ Настройки

Мод **не требует настройки**. Все цифры (`скорость потока`, `длительность цикла`, `количество циклов`) считываются **автоматически** из данных игры при наведении на гейзер.
